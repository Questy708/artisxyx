# 33. TECHNOLOGY SETUP GUIDE

**xHansa Fellowship — Technology Setup Guide**
**Complete before Crucible Day 1**

---

## OVERVIEW

This guide covers every piece of technology you'll use during your deployment. Set it all up on Day 1–2 (Orientation). The Crucible starts at 06:00 on Day 3 — if your tech isn't working by then, you're already behind.

---

## 1. DEVICES

### 1.1 Personal Laptop (Required)
Bring your own laptop. Minimum specs:
- **OS:** macOS 12+ / Windows 10+ / Ubuntu 22.04+
- **RAM:** 8GB minimum (16GB recommended for Builder/Hustler Archetypes)
- **Storage:** 256GB minimum
- **Browser:** Chrome or Firefox (latest version)

**Setup:**
- [ ] Install all OS updates before arrival
- [ ] Install a password manager (Bitwarden or 1Password — the League provides a Bitwarden team account)
- [ ] Install Zoom or Google Meet (for external partner calls)
- [ ] Install Signal (for encrypted backup communication)
- [ ] Ensure full-disk encryption is enabled (FileVault / BitLocker / LUKS)

### 1.2 Personal Phone (Required)
- **OS:** iOS 15+ or Android 11+
- **Apps to install:**
  - [ ] Slack (primary communication)
  - [ ] TownSquare (PWA — add to home screen)
  - [ ] Signal (backup communication)
  - [ ] Google Authenticator or Authy (for 2FA on all League accounts)
  - [ ] Your bank's app (for stipend deposits)

**Note:** Your phone will be held by Crucible instructors during scenario sessions. It is returned during meals and after the day's scenarios. Plan accordingly (family contacts, personal admin during meal breaks only).

### 1.3 Neural Link Device (Provided)
- Wrist-worn biometric monitor
- **Setup:** Done at medical screening (Day 1, 15:00). Device is fitted, activated, and paired to your profile.
- **Charging:** Charge during meals and shower (30–45 min charge = 24+ hrs battery). Never go more than 2 hours without it.
- **Care:** Waterproof (shower-safe, not swim-safe). If it malfunctions, report to the Hub Manager immediately — a replacement will be issued within 4 hours.

### 1.4 XEmbassy Equipment (On-Site)
Shared equipment available at the XEmbassy (sign out via the equipment log):
- Prototyping tools (3D printers, CNC, electronics lab — if your ProtoCo requires)
- Field testing equipment (sensors, meters — Field-specific)
- Backup laptops (limited, emergency use only)
- Printers / scanners (for documents)

## 3. SLACK SETUP (Primary Communication Tool)

Slack is your primary communication channel during deployment. Set it up properly on Day 1.

### 3.1 Channels You'll Be In

| Channel | Purpose |
|---------|---------|
| `#pod-[your-pod-id]` | Your Pod's private channel (10 members + Pod Leader) |
| `#cohort-[N]` | Cohort-wide announcements and discussion |
| `#field-[your-field]` | Your Civilizational Field community (cross-Pod) |
| `#commons-contributions` | Where you post weekly Commons contributions |
| `#announcements` | Strategic Command announcements (read-only) |
| `#support` | Tech support, logistics questions, general help |
| `#medical` | Confidential medical support (DM the medical team) |

### 3.2 Slack Configuration

- [ ] **Notifications:** Enable push notifications on mobile. Set "Do Not Disturb" to 22:00–05:30.
- [ ] **Display name:** [First Name] [Last Initial] — [Archetype] (e.g., "Amara O. — Builder")
- [ ] **Profile photo:** Clear headshot
- [ ] **Status:** Set your current phase (e.g., "🟠 Crucible — Week 2")
- [ ] **Time zone:** Set to the XEmbassy's time zone

### 3.3 Slack Etiquette

- **Response time:** Within 4 hours during working hours (06:00–21:00). After hours, respond by 08:00 the next day.
- **Direct messages:** Use for personal/sensitive matters. Use channels for everything else (transparency).
- **Threads:** Keep discussions in threads. Don't clutter the main channel.
- **Urgency:** Use `@here` for urgent Pod matters. Use `@channel` only for true emergencies. Use `@pod-leader` to flag your Pod Leader.
- **Strategic Command:** Never DM Strategic Command directly. Go through your Pod Leader.

## 7. SECURITY

### 7.1 Password Hygiene
- Use Bitwarden for all League accounts. Never reuse passwords.
- 2FA on every account (Google Authenticator).
- Never share credentials — not even with your Pod. Each person has their own access.

### 7.2 Device Security
- Full-disk encryption enabled (Section 1.1).
- Screen lock after 5 minutes of inactivity.
- Don't leave devices unattended in public areas.
- Report lost/stolen devices to the Hub Manager immediately (access will be revoked remotely).

### 7.3 Data Security
- League data stays on League devices/accounts. Don't email Confidential Information to personal accounts.
- Don't install unverified software on devices used for League work.
- Don't use public Wi-Fi without a VPN (the League provides a VPN — install it).

### 7.4 Comms Security
- Slack is for internal communication (moderately confidential).
- Signal is for sensitive communication (encrypted).
- Email is for official/external communication.
- Never discuss Confidential Information on phone calls in public spaces.
